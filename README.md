# IT Support Troubleshooting Lab

## What this is
This repo contains real-world style troubleshooting notes based on common IT support issues like internet outages, DNS problems, and network misconfigurations.

The goal is to document how issues are identified and resolved in a structured but realistic way.

## What I practiced here
- Figuring out why devices say “connected but no internet”
- Checking IP and DNS issues
- Basic network troubleshooting steps
- Writing clear support notes like in a helpdesk environment

## Common issues covered
- No internet even though Wi-Fi is connected
- DNS not resolving websites
- Incorrect gateway settings
- Slow or unstable network connection

## Format used
Each issue is written like a support ticket:
- What the user experienced
- What I checked
- What caused it
- How it was fixed
