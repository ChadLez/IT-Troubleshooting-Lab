# Project Summary

This project was built to practice realistic IT troubleshooting and technical documentation.

The focus was on:
- Diagnosing common network issues
- Understanding IP and DNS behavior
- Applying structured troubleshooting methods
- Writing support-style notes clearly

Main concepts practiced:
- DHCP
- DNS
- Gateway troubleshooting
- Connectivity testing

This lab helped reinforce how common IT support problems are approached in real environments.
