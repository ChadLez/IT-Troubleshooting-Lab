# DNS Troubleshooting Notes

## Common symptoms
- Websites not loading
- Browser connection errors
- Internet appears connected but domains fail

## Checks performed
- Ping tests
- DNS lookup tests
- DNS server replacement

## Fixes used
- Changed DNS to 8.8.8.8
- Cleared DNS cache
- Restarted router/network adapter

## Main takeaway
DNS issues can sometimes look like full internet outages even when connectivity still exists.
