# Network Diagram (Simple View)

Laptop → Wi-Fi Router → Internet Service Provider → Internet → DNS Server (8.8.8.8)

---

## How data flows

1. Laptop connects to Wi-Fi router
2. Router assigns IP address via DHCP
3. Router forwards traffic to ISP
4. ISP routes traffic to the internet
5. DNS server translates domain names (like google.com) into IP addresses
