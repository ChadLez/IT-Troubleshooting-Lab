# Network Diagram (Simple View)

Laptop
  ↓
Wi-Fi Router (DHCP assigns IP address)
  ↓
Internet Service Provider (ISP)
  ↓
Internet
  ↓
DNS Server (8.8.8.8 resolves domain names)

---

## How data flows

1. Laptop connects to Wi-Fi router
2. Router assigns IP address using DHCP
3. Traffic is sent through ISP
4. ISP routes traffic to the internet
5. DNS converts domain names (like google.com) into IP addresses
